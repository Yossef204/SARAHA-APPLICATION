# 🚀 Anonymous Messaging Backend System

A scalable and secure backend system inspired by [Sarahah App Official Website](https://saraha.com?utm_source=chatgpt.com), built to simulate how real-world backend applications are structured and maintained.

This project focuses on authentication, anonymous messaging, security, caching, file handling, and clean backend architecture using modern backend technologies.

---

# ✨ Features

## 🔐 Authentication & Security

* JWT Authentication
* Secure Password Hashing
* Email Verification System
* Google Sign-In Integration
* Protected Routes
* Validation & Error Handling

## 💬 Anonymous Messaging

* Send anonymous messages to users
* Receive and manage messages securely
* Real-world backend logic implementation

## ⚡ Performance & Optimization

* Optimized API Responses

# ⚙️ Installation

## 1️⃣ Clone the repository

```bash
git clone YOUR_REPOSITORY_LINK
```

## 2️⃣ Navigate to the project folder

```bash
cd PROJECT_NAME
```

## 3️⃣ Install dependencies

```bash
npm install
```

## 4️⃣ Create environment variables

Create a `.env` file and add:

```env
PORT=
MONGO_URI=
JWT_SECRET=
REDIS_URL=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
EMAIL_USER=
EMAIL_PASS=
```

## 5️⃣ Run the server

```bash
npm run dev
```
